# Quiz Quest Arena: 맞춤 퀴즈 — 독립 복제본

원본 공개 프런트엔드: https://quiz-math-6-1-1.vercel.app/

공개 Vite/React/Phaser 정적 자산을 바탕으로 교사 맞춤 문제 세트를 등록할 수 있게 확장한 교육용 게임입니다. 원본의 저작권·표시·Phaser 고지는 유지합니다.

## 기능

- 6학년 1학기 「분수의 나눗셈」 300문항
- 교사 모드 잠금 해제 후 맞춤 문제 세트 등록
- ZEP Quiz 드래그 복사 텍스트 붙여넣기 가져오기
- CSV/XLSX 엑셀 파일 가져오기
- 선다형, 단답형, OX형 문제 저장
- 수동 문제 추가 및 정답 보정
- Phaser 3 생존형 자동공격 게임
- 키보드와 모바일 조이스틱
- 난이도 3단계
- 퀴즈 정답에 따른 무기 획득·강화
- 브라우저 localStorage 기반 오프라인 랭킹

## 게임 모드 (3종)

- **퀘스트 서바이버** (`index.html`): 원본 기반 생존형 자동공격 게임
- **타워 디펜스** (`tower.html`): 경로를 따라 밀려오는 몹을 히어로 타워(궁수·마법사·검사)로 방어. 웨이브 클리어마다 보너스 퀴즈로 코인 획득, 10웨이브 + 보스 2종
- **라스트 워: 서바이벌** (`lastwar.html`): 세로 레인 슈터. 부대가 자동 사격하며, 내려오는 **정답 게이트**를 통과하면 부대 증원(오답 시 절반). 좌우 키·드래그 조작

두 신규 모드는 `assets/games/`의 Canvas2D 엔진으로 구현되었고, 교사 등록 문제(`quiz_survivor_custom_questions`)가 있으면 기본 300문항 대신 그 세트를 사용합니다.

Firebase 환경값은 원본 공개 빌드에서도 비활성 상태이며, 이 복제본 역시 별도 원격 랭킹 서버나 개인정보 전송을 추가하지 않습니다.

교사 비밀번호는 Vercel 환경변수 `TEACHER_PASSCODE`로 관리합니다. 클라이언트 코드에는 비밀번호를 저장하지 않습니다.

## 문제 등록

교사 모드에서 다음 형식을 지원합니다.

- 붙여넣기: ZEP Quiz에서 문제 목록을 드래그해 복사한 텍스트
- 엑셀/CSV: `유형`, `문제`, `정답`, `선택지1`, `선택지2`, `선택지3`, `선택지4`, `해설`, `난이도` 열
- 수동 입력: 화면에서 문제, 보기, 정답을 직접 작성

붙여넣기 가져오기는 `유사 문제 생성`, `question-image` 같은 장식 문구를 자동으로 제거합니다. 복사본에 정답이 확실하지 않은 문제는 답칸을 비워 두므로 교사가 확인해서 입력해야 합니다.
